# BP-Tracker
Local-only Windows blueprint and material tracker.
# BP Tracker v0.1 Beta

Initial beta release.

## Features

- Blueprint inventory tracking
- Need / Hold / Trade status borders
- Material tracking
- Workbench material view
- In-game order and alpha sort
- Manual BP update ZIP importer
- Local-only data storage
- Windows installer and uninstaller support

## Download

Download and run:

BP_Tracker_Setup_v0.1_Beta.exe

## Notes

- Windows only.
- This is a beta release.
- The installer is unsigned, so Windows SmartScreen may show an Unknown Publisher warning.
- User data is stored locally in `%LOCALAPPDATA%\BP Tracker`.
- No login, no cloud sync, no telemetry, no data collection.
